  • Automated Switch For Clothing Iron

  • Analog Heartbeat Sensor

    This project presents the design of an analog heartbeat sensor that utilizes PPG to measure changes in blood volume at the fingertip using an IR LED and photodiode in reflective mode.

    An Arduino microcontroller samples the conditioned signal and calculates BPM, which is displayed on an OLED screen for real-time monitoring.

    Experiences
    • Designed and implemented an analog heartbeat sensor using PPG principles

  • Two-Way Digital Paging System Using SDR

    This project focuses on the design and simulation of a two-way digital paging system using Software Defined Radio (SDR) concepts. The system enables reliable wireless transmission of short text messages between two users, incorporating essential digital communication techniques such as modulation, framing, addressing, error detection, and acknowledgments.

    This project provides practical exposure to digital communication system design, SDR architectures, and GNU Radio flowgraph development, while laying the foundation for a future hardware implementation using BladeRF in the final evaluation.

    Experiences
    • Designed SDR-based communication system using GNU Radio
    • Implemented modulation, framing, CRC, and ACK mechanisms
    • Analyzed protocol behavior using SDR simulation
